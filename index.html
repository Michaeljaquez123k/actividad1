Funcionamiento Básico de CSS:

CSS funciona aplicando reglas de estilo a los elementos de una página web. Estas reglas se escriben en archivos CSS y se pueden aplicar a los elementos HTML mediante selectores.

cluir CSS en un archivo HTML

En línea: Aplicar estilos directamente en los elementos HTML mediante el atributo “style”.

Interno: Incluir un bloque de estilos dentro del elemento

Definición de una regla CSS:

Una regla CSS define cómo deben aplicarse los estilos a ciertos elementos. Está compuesta por un selector y un bloque de declaración.

Elemento de una regla CSS:

Regla: Es la combinación de un selector y un bloque de declaración.

Selector: Es el patrón que selecciona los elementos HTML que se desea estilizar.

Bloque de declaración:

Es el conjunto de declaraciones de estilo que se aplican a los elementos seleccionados. Está encerrado entre llaves {}.

Propiedad: Es un aspecto específico del estilo que se desea definir, como color, font-size, margin, etc. Se escribe antes de los dos puntos en una declaración de estilo.

Valor: Es el valor asignado a una propiedad. Se escribe después de los dos puntos en una declaración de estilo.

-Medios CSS

Los medios CSS se refieren a las reglas que permiten aplicar estilos específicos según el tipo de dispositivo o las características del entorno de visualización. Se utilizan con las consultas de medios (media queries) para hacer que las páginas web sean más responsivas.

Comentarios CSS: Los comentarios en CSS se utilizan para añadir notas dentro del código. Los comentarios no afectan al estilo visual y se escriben entre /* y */.
